# 📊 Job Market Analysis and Forecasting

## 📌 Overview

This project is an interactive Power BI dashboard that analyzes job market trends, in-demand skills, salary patterns, AI impact on jobs, and future hiring trends. The data was prepared using **Microsoft Excel**, while **Power BI** and **DAX** were used to build dynamic visualizations and interactive reports.

## 🛠️ Tools Used

* Microsoft Excel
* Power BI
* DAX (Data Analysis Expressions)

## 📈 Dashboard Pages

* 📊 Job Market Overview
* 💼 Skills Demand Analysis
* 💰 Salary Analysis
* 🤖 AI Impact Analysis
* 📉 Predictive Analysis (Forecasting)

## ✨ Key Features

* Interactive Dashboard
* Dynamic KPIs
* Salary Insights
* In-Demand Skills Analysis
* AI Impact Analysis
* Hiring Trend Forecasting
* Interactive Filters & Slicers
